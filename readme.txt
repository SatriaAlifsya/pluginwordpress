This plugin was created to help admins manage content automatically, choose which post type to display, and filter ratings from reviews taken.

== Description ==

Auto Get Reviews Combined is a WordPress plugin that helps admins manage content efficiently. It fetches reviews from an API and includes an auto-generate post feature to streamline content creation. Version 1.0 is the initial release and will be developed further in the future.

== External services ==

This plugin connects to Google Maps API to fetch reviews, ratings, and photos. The data sent includes:
- Google Place ID
- Google API Key

Terms of Service: https://cloud.google.com/maps-platform/terms
Privacy Policy: https://policies.google.com/privacy

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings via the provided admin panel.
